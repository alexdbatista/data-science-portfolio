## â“ Troubleshooting

**Q: "FileNotFoundError: data/processed_biomarkers.csv not found"**  
**A:** Run `01_Signal_Processing_and_EDA.ipynb` first to generate the training data.

**Q: "FileNotFoundError: models/glucose_rf_v1.pkl not found"**  
**A:** Run `02_Model_Training_and_Clinical_Evaluation.ipynb` to train and save the model.

**Q: Can I use my own CGM data?**  
**A:** Yes! Replace the data generation cell in notebook 01 with your own data loading logic. Ensure the schema matches the expected column names and ranges above.
